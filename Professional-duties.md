As part of my job, I was involved in the creation of the web application for Beraria H. I used the Moscow method and Double Diamond framework, created a thorough definition of done, and carefully developed user stories. By utilising collaborative tools like Clickup and Miro board, I was able to manage project tasks effectively. I took on a design challenge, shaping the Beraria H UI while following the style guide and communication guidelines that had been set forth. Working at Beraria H improved my comprehension of professional responsibilities, which include a broad range of tasks from conception to completion.
